# Setup

### Installing Haskell -- FOR MAC 
- `curl -sSL https://get.haskellstack.org/ | sh`
- `stack exec ghci`

### Running Haskell on your terminal
- Type `ghci` on your terminal 
- To quit, just type `:quit` 

### Using Haskell in an editor 
- .hs files
